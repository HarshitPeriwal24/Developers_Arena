# 🎓 Student Grade Calculator

## 📌 Description
This Python program takes student name and marks as input and calculates the grade based on predefined criteria. It also provides encouraging feedback messages.

## 🛠️ Features
- Input validation (marks between 0-100)
- Uses functions for clean code
- Implements if-elif-else grading logic
- Handles invalid input using while loop
- Displays user-friendly output

## 📝 Grading System
- A: 90-100
- B: 80-89
- C: 70-79
- D: 60-69
- F: 0-59

## ▶️ How to Run
```bash
python grade_calculator.py
